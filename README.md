# Data-Pipelines

A collection of data engineering and Python projects, exercises, and examples used for learning and class assignments.

## Overview
- Focus areas: data analysis, database integration, object-oriented programming, GUI apps with Tkinter, and small ML examples.

## Repository Structure

| Folder | Description | Notable files |
|---|---|---|
| Inventory | Miscellaneous notes and resources | README.md |
| DataBases-BackEnd | Database examples, architecture notes, and ML scripts | IrisDatasetLogisticRegressionNumPy.py, IrisDatasetLogisticRegressionSklearn.py, DB/Architecture.md |
| NumPy | Numerical examples, data analysis and notebooks | rocket_launch.py, rocket.py, new_york_housing/ |
| OOP W1.2 | Object-oriented programming exercises (week 1.2) | README.md |
| OOP_tasks_w0 | OOP tasks (week 0): small projects and CLI examples | Coin Accept/, Counter/, Temperature Convertor/, Weight Conversion/ |
| OOP_tasks_w1 | OOP tasks (week 1): design patterns and exercises | Abstract Base Class/, Polymorphic Smart Home/, Encapsulation CryptoWallet/ |
| Serialization | Serialization examples and notes | SERIALIZATION.md |
| Task Productivity Application (tkinter) | A small Tkinter-based productivity app | database.py, user_interface.py |
| Tkinter | Collection of Tkinter GUI examples and demos | dashboard.py, main.py, database.py, dropdown.py |

## Quick Links
- New York Housing Data Analysis: NumPy/new_york_housing

## How to use
- Browse the folders above to find the example or exercise you need.
- Run individual Python scripts with your Python 3 interpreter, e.g. `python path/to/script.py`.
